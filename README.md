# Comprehensive Menu

- Multi-level.
- Responsive.
- Recursive.
- Data from array.

## Project setup
### Init react-app
`npx create-react-app comprehensive-menu --template typescript`

`--template typescript` is optional for typescript project

### Install react-router-dom
`npm install --save react-router-dom`

### Start react-app
```
cd comprehensive-menu
npm start
```
